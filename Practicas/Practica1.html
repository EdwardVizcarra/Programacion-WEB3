Layouts

Existen 6 layouts importantes para representar el posicionamientos de las cosas que coloquemos en nuestra pagina

  1. Block layout

    2. diseñado para presentar documentos. El block layout contiene caracteristicas de cocumento centrado como la capacidad de flotar (float) elementos o distribuirlos en multiples columnas

  2. inline layout

    3. diseñado para presentar texto

  3. table layout

    4. diseñado para presentar tablas

  4. positioned layout

    5. diseñado para posicionar elemtnos sin demaiada interracion con otros elementos

  5. flexible box layout

    6. diseñado para presentar paginas complejas que pueden redimensionarse de forma fluida

  6. grid layout

    7. diseñado para presentar elemtnos relativos a una cuadricula fija
